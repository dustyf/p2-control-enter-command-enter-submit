== Description ==
This plugin requires the P2 theme.  This will add the ability to submit posts and comments by using the keyboard shortcut of command-enter or control enter.

== Installation ==
= Plugin Repository =

Search the WordPress Plugin Repository and click install

= Upload =

1. Download Zip File
2. Go to Add New Plugins
3. Upload Zip File
4. Activate Plugin

= Manual Installation =

1. Unzip files
2. Upload Plugin Folder to Plugins Directory
3. Activate Plugin
